# Financial Import MVP

This is a local Node.js + Express + SQLite starter app for turning `MyFinancials.xlsx` into a browsable application.

## Run locally

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Core files

- `app.js`
- `package.json`
- `views/index.ejs`
- `views/import-detail.ejs`
- `public/styles.css`
